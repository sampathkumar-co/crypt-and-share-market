# Risk Policy

- Version 1 is paper trading only and must not place live trades.
- No leverage, futures, or options live trading initially.
- No withdrawal permission should ever be requested for future crypto API keys.
- No real wallet, exchange trading, or broker order API integration in v1.
- No guaranteed-profit claims in UI, reports, or documentation.
- Use long paper-testing periods before considering tiny live capital.
- A kill switch, audit logs, strict API permissions, and deployment review are required before any future live trading.
- Indian tax and brokerage values are estimates only and are not tax advice.
